Домашнее задание по теме "Протоколы и модели сетевого взаимодействия"

Цель домашнего задания - сформировать практические навыки разработки серверных приложений с соблюдением принципов REST API: научиться проектировать конечные точки (endpoints), корректно обрабатывать HTTP-методы и статус-коды, реализовывать интерфейсы через фабричный паттерн, а также отработать профессиональный рабочий процесс — от написания тестов и соблюдения код-стайла до публикации кода через Pull Request с оперативной реакцией на ревью.


Разработайте HTTP-сервер с API для хранения данных по ключам с поддержкой создания, чтения и удаления записей по заданному HTTP-интрефейсу.
Подготовьте готовое решение в виде форка целевого репозитория на github.com, и отправьте код на проверку через Pull Request (PR).
PR должeн проходить все автоматические провекри - тесты и код-стайлы. Своевременно реагируйте на замечания к PR.

Критерии оценивания домашнего задания: 

1. За выполение основного задания
(все 5 пунктов) - 6 баллов
2. За первое бонусное +1 балл
3. За второе бонусное +3 балла

Итого: 10 баллов
